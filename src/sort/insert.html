插入排序

3.2 插入排序（Insertion Sort）

插入排序又为分为 直接插入排序 和优化后的 拆半插入排序 与 希尔排序，我们通常说的插入排序是指直接插入排序。

一、直接插入

通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

// 插入排序
const insertionSort = arr => {
        console.time("insertionSort")
        let len  = arr.length
        if(len <=1) return
        let preIndex,current
        for(let i = 1;i<len;i++){
            preIndex = i -1
            current = arr[i]
            while(preIndex >=0 && arr[preIndex] > current){
                arr[preIndex+1] = arr[preIndex]
                preIndex --
            }
            if(preIndex+1 != i){
                arr[preIndex+1] = current
            }
        }
        console.timeEnd("insertionSort")
        console.log(arr)
    }
    const arr4 = [98, 96, 94, 93, 93, 93, 93, 92, 92, 91, 91, 89, 88, 87, 87, 86, 86, 85, 85, 84, 82, 81, 79, 79, 79, 76, 76, 75, 73, 70, 70, 66, 64, 64, 63, 56, 56, 55, 54, 53, 52, 51, 50, 49, 47, 47, 46, 45, 45, 45, 44, 44, 44, 41, 41, 39, 39, 38, 38, 35, 35, 33, 32, 32, 32, 31, 29, 28, 26, 25, 24, 23, 21, 21, 21, 21, 21, 20, 20, 19, 15, 15, 13, 13, 12, 11, 10, 9, 9, 8, 8, 7, 4, 4, 4, 3, 3, 2, 0, 0];
    insertionSort(arr4);
//insertionSort: 0.760986328125ms

分析

  • 第一，插入排序是原地排序算法吗 ？ 插入排序算法的运行并不需要额外的存储空间，所以空间复杂度是 O(1)，所以，这是一个原地排序算法。
  • 第二，插入排序是稳定的排序算法吗 ？ 在插入排序中，对于值相同的元素，我们可以选择将后面出现的元素，插入到前面出现元素的后面，这样就可以保持原有的前后顺序不变，所以插入排序是稳定的排序算法。
  • 第三，插入排序的时间复杂度是多少 ？ 最佳情况：T(n) = O(n)，当数据已经是正序时。 最差情况：T(n) = O(n2)，当数据是反序时。 平均情况：T(n) = O(n2)

二、拆半插入？？

插入排序也有一种优化算法，叫做拆半插入。

思想

折半插入排序是直接插入排序的升级版，鉴于插入排序第一部分为已排好序的数组，我们不必按顺序依次寻找插入点，只需比较它们的中间值与待插入元素的大小即可。

步骤

  • 取 0 ~ i-1 的中间点 ( m = (i-1) >> 1 )，array[i] 与 array[m] 进行比较，若 array[i] < array[m]，则说明待插入的元素 array[i] 应该处于数组的 0 ~ m 索引之间；反之，则说明它应该处于数组的 m ~ i-1 索引之间。
  • 重复步骤 1，每次缩小一半的查找范围，直至找到插入的位置。
  • 将数组中插入位置之后的元素全部后移一位。
  • 在指定位置插入第 i 个元素。

注：x >> 1 是位运算中的右移运算，表示右移一位，等同于 x 除以 2 再取整，即 x >> 1 == Math.floor(x/2) 。

// 折半插入排序
const binaryInsertionSort = array => {
    console.time("binaryInsertionSort")
    const len = array.length;
    if (len <= 1) return;

    let current, i, j, low, high, m;
    for (i = 1; i < len; i++) {
        low = 0;
        high = i - 1;
        current = array[i];

        while (low <= high) {
            m = (low + high) >> 1; 
            if (array[i] >= array[m]) {
                low = m + 1;
            } else {
                high = m - 1;
            }
        }
        for (j = i; j > low; j--) {
            array[j] = array[j - 1];
        }
        array[low] = current;
    }
    console.timeEnd("binaryInsertionSort")
    console.log(array)
};
const arr5 = [98, 96, 94, 93, 93, 93, 93, 92, 92, 91, 91, 89, 88, 87, 87, 86, 86, 85, 85, 84, 82, 81, 79, 79, 79, 76, 76, 75, 73, 70, 70, 66, 64, 64, 63, 56, 56, 55, 54, 53, 52, 51, 50, 49, 47, 47, 46, 45, 45, 45, 44, 44, 44, 41, 41, 39, 39, 38, 38, 35, 35, 33, 32, 32, 32, 31, 29, 28, 26, 25, 24, 23, 21, 21, 21, 21, 21, 20, 20, 19, 15, 15, 13, 13, 12, 11, 10, 9, 9, 8, 8, 7, 4, 4, 4, 3, 3, 2, 0, 0];
binaryInsertionSort(arr5);
// binaryInsertionSort: 0.480712890625ms

注意：和直接插入排序类似，折半插入排序每次交换的是相邻的且值为不同的元素，它并不会改变值相同的元素之间的顺序，因此它是稳定的
