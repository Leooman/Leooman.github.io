| 位运算 | 说明                                                         |
| ------ | ------------------------------------------------------------ |
| &      | And按位与 有零为零                                           |
| \|     | Or按位或 有一为一                                            |
| ^      | Xor 按位异或 相同为零 不同为一                               |
| ~      | Not 按位取反                                                 |
| <<     | 左移                                                         |
| >>     | 有符号右移                                                   |
| >>>    | 无符号右移，左边补零，运算结果为一个无符号32位整数（右移32位变为原值） |

> 判断一个数是不是偶数

```js
x & 1 === 0
```

> 交换两个数

```js
//使用临时变量
temp = a
a = b
b = temp
//不使用临时变量
a ^= b
b ^= a
a ^= b
```

> 取相反数

```js
~x + 1
```

> 判断是不是2的幂

```js
n & (n-1)
```

> 二进制中1的个数

```js
let count = 0
while(x){
  ++count
  x = x & (x - 1)
}
```

> 对2的n次方取余

```js
x & (1 << n - 1)
```
