队列

队列 Queue

普通队列

  • 队列是遵循 FIFO（First In First Out，先进先出）原则的一组有序的项。
  • 队列在尾部添加新元素，并从顶部移除元素。
  • 最新添加的元素必须排在队列的末尾。
  • 队列只有 入队 push() 和出队 pop()。
class Quque {
    constructor(){
        this.quque = []
    }
    // 队列尾部追加元素
    enquque(element){
        this.quque.push(element)
    }
    // 返回移除队列的第一个元素
    dequque(){
        return this.quque.shift()
    }
    // 返回队列的第一个元素
    front(){
        return this.quque[0]
    }
    // 判断是否为空队列
    isEmpty(){
        return this.quque.length === 0
    }
    // 获取队列长度
    size(){
        return this.quque.length
    }
    // 清空队列
    clear(){
        this.quque = []
    }
    // 打印队列
    print(){
        console.log(this.quque.toString())
    }
}

优先队列

优先队列中元素的添加和移除是依赖优先级的。

  • 最小优先队列
  • 最大优先队列

最小优先队列是把优先级的值最小的元素被放置到队列的最前面（代表最高的优先级）。 比如：有四个元素："John", "Jack", "Camila", "Tom"，他们的优先级值分别为 4，3，2，1。 那么最小优先队列排序应该为："Tom"，"Camila"，"Jack"，"John"。

最大优先队列正好相反，把优先级值最大的元素放置在队列的最前面。 以上面的为例，最大优先队列排序应该为："John", "Jack", "Camila", "Tom"。

// 最小优先队列，和普通队列的不同在于元素的插入方式
class MinPriorityQueue {
    constructor(){
        this.quque = []
    }
    enquque(element,priority){
        let ququeElement = {
            element:element,
            priority:priority
        }
        if(this.isEmpty()){
            this.quque.push(ququeElement)
        }else{
            let added = false
            for(let i = 0;i < this.size();i++){
                if(ququeElement.priority < this.quque[i].priority){
                    this.quque.splice(i,0,ququeElement)
                    added = true
                    break
                }
            }
            if(!added){
                this.quque.push(ququeElement)
            }
        }
    }
}

循环队列

关键是：确定好队空和队满的判定条件

// 实现击鼓传花
const CircleQuque = (namelist,number) => {
    let quque = new Quque()
    for(let i of namelist){quque.enquque(i)}
    let eliminated = null
    while(quque.size() > 1){
        for(let i = 0;i < number;i++){
            quque.enquque(quque.dequque())
        }
        eliminated = quque.dequque()
        console.log(`${eliminated} 在击鼓传花中被淘汰！`)
    }
    return quque.dequque()
}

// 测试
var nameList = ["John", "Jack", "Camila", "Ingrid", "Carl"]
var winner = CircleQuque(nameList, 10)
console.log(`最后的胜利者是：${winner}`)
